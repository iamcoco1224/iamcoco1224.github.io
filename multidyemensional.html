Multidyemensional Goods
Information Design

Objective Select an web site that needs to be redesigned. Create a Web site information structure based on user tests and information analysis. Focus on using information to curate and possibly create content.

Background Multidyemensional Goods is an artist-owned small business which focuses on selling hand-made tie-dye clothes. It provides membership, online payment, and newsletter subscription. Users can shop by collections and categories as well as search the product directly. The owner is also the key stakeholder whose name is Andrea Buchanan. She is a graduate student from SCAD and this website is also her M.A. final project. She runs and designs this website through Shopify, an e-commerce company that offers online retailers a suite of services "including payments, marketing, shipping and customer engagement tools to simplify the process of running an online store for small merchants.‚Äù
